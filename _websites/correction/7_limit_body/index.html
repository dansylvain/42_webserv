For testing the Body limit, launch another instance of webserv with the following config

		
			./webserv _websites/correction/7_limit_body/limit_body.ini

NOTA BENE : In this config, the limit is setup at 10, and uploads to uploads/ base directory !

The goal is then to try with a body size == 10, < to 10 and > to 10

You can open the console, and test the following CURL commands :

Command with body of size < 10

		
			curl -i -s -X POST -H "Content-Type: text/plain" -d '123456789-10' http://localhost:3008

Command with body of size == 10

		
			curl -i -s -X POST -H "Content-Type: text/plain" -d '1234567890' http://localhost:3008

Command with body of size > 10

        
            curl -i -s -X POST -H "Content-Type: text/plain" -d '123456789' http://localhost:3008
