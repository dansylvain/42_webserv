🔴 RED PILL : CURL 🔴

So you choosed the Red Pill...


The goal here is to upload files with CURL.

You will try 3 curl commands : with a body size == 10, < to 10 and lastly > to 10



You can open the terminal, and test the following CURL commands :

Command with body of size > 10

( 🖱️CLICK ME ) curl -i -s -X POST -H "Content-Type: text/plain" -d '123456789-10' http://localhost:3008

Command with body of size == 10

( 🖱️CLICK ME ) curl -i -s -X POST -H "Content-Type: text/plain" -d '1234567890' http://localhost:3008

Command with body of size < 10

( 🖱️CLICK ME ) curl -i -s -X POST -H "Content-Type: text/plain" -d '123456789' http://localhost:3008
